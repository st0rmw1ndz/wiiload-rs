# wiiload-rs

Wiiload implementation in Rust.

## Usage

```
Wiiload implementation in Rust.

Usage: wiiload.exe <IP> <PATH>

Arguments:
  <IP>    IP address
  <PATH>  Path to file

Options:
  -h, --help     Print help
  -V, --version  Print version
```
